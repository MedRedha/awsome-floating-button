## Build & Run

#### Prerequisite

```bsh
$ yarn i -g nodemon
$ yarn i
```


#### Development Mode

```bsh
$ yarn start
$ yarn test
```

#### Production Mode

```bsh
$ yarn build
$ yarn eject
```
